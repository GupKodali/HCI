How to Run This Website Using VS Code + Live Server

This project is a multi-page HTML website.
To run it locally with automatic reloading, follow the instructions below.

Requirements

Visual Studio Code (free)

Live Server extension for VS Code

Zip Folder: 
How to Run the Website
1. Install Visual Studio Code

Download and install VS Code from:
https://code.visualstudio.com/

2. Install the “Live Server” Extension

Open VS Code

Click the Extensions icon on the left

Search for: Live Server

Install the extension that is most popular

3. Open the Project Folder

In VS Code, go to File → Open Folder

Select the Login folder that contains the Login.html 

4. Start Live Server

You can launch it in either of these ways:

Option A — Right-click method

Right-click on Login.html

Select “Open with Live Server”

Option B — Go Live button

Click the “Go Live” button in the bottom-right corner of VS Code

5. View the Website

Your browser will open automatically at a URL like:

http://127.0.0.1:5500/
